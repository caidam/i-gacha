Build a birthday gachapon

Pick a vibe, add some gifts, and you'll get a shareable link to send. The whole lottery lives in the link itself — no accounts, no servers.

What is this?

i-gacha is a personal gachapon you can build and send to someone. The lottery — gifts, theme, password, dud chance — is encoded directly into the share link, so anyone with the link plays your version, even if they've never seen this site before.

Nothing is uploaded anywhere. The page runs entirely in your browser; the link IS the storage. With a session password, gifts are encrypted in the link itself, so even copying the URL into a logbook reveals nothing.

Step 1 — required

Admin password

After sharing, you can come back any time: tap the lottery title 5 times in a row, then enter this password to edit gifts and copy a fresh link. Don't lose it — there's no recovery.

Step 2 — optional

Session password

If set, the recipient enters this to open the lottery, and the gifts are encrypted in the link itself. Leave blank for an open link anyone can play.

Step 3

Pick a vibe

The theme styles the whole experience.

Step 4

Dud capsules

Some pulls come up empty for suspense. Set 0% if every pull should be a guaranteed win.

Step 5

Gifts

Add anything: a dinner reservation, a playlist, a place, a joke, an IOU. Optional links open when revealed. Tap ✎ to edit, ☆ to mark a special "jackpot" prize that glows in the dome. Set a date and time to lock a gift until then — perfect for advent-style surprises. Lock times use the recipient's local timezone.
